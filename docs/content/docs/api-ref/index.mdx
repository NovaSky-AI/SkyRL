# API Reference

## SkyRL

SkyRL is the core reinforcement learning framework. The **SkyRL-Train** backend provides all the training abstractions — the Trainer, Data Interface, Generator, Entrypoints, and more.

- [**Trainer**](/docs/api-ref/skyrl/skyrl-train/trainer) — Trainer class, dispatch, and actor APIs
- [**Data Interface**](/docs/api-ref/skyrl/skyrl-train/data) — Training data types (`GeneratorInput`, `GeneratorOutput`, etc.)
- [**Generator**](/docs/api-ref/skyrl/skyrl-train/generators) — Generator and inference engine APIs
- [**Entrypoint**](/docs/api-ref/skyrl/skyrl-train/entrypoints) — Training and evaluation entrypoints
- [**Algorithm Registry**](/docs/api-ref/skyrl/skyrl-train/registry) — Custom algorithm function registration
- [**Environment Variables**](/docs/api-ref/skyrl/skyrl-train/env-vars) — Configuration via environment variables
